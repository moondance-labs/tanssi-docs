# Construye tu red

Lanzar tu red con tecnología Tanssi te da la flexibilidad de elegir entre plantillas preconfiguradas o soluciones totalmente personalizadas. Ya sea que busques simplicidad o control preciso, Tanssi proporciona las herramientas y la guía para hacer realidad tu visión. Esta sección cubre todo, desde la selección de una plantilla hasta la personalización de módulos, pruebas e implementación de tu red.

## La ruta hacia la implementación

Implementar una red con Tanssi puede ser sencillo o altamente personalizable, según tus requisitos. El siguiente diagrama proporciona un desglose claro de estas opciones para guiar tu decisión.

```mermaid
flowchart TD
    Start[Elige una plantilla]
    Start --> Path1[La plantilla es suficiente]
    Start --> Path2[La plantilla necesita personalización]

    Path1 --> Deploy1[Implementar a través de Tanssi dApp]

    Path2 --> Fork[Copia el repositorio de Tanssi]
    Fork --> Customize[Personaliza tu red]
    Customize --> Test[Ejecuta pruebas]
    Test --> Deploy2[Implementar a través de Tanssi dApp]
```

Ahora que has explorado las rutas de implementación, el siguiente paso es [aprender más sobre cada plantilla](/builders/build/templates) para que puedas elegir la que mejor se adapte a tus necesidades.
