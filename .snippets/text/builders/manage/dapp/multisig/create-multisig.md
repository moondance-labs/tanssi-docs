To create a new multisig, click **Create Multisig Account** and follow the steps in the modal windows that appear:

1. Set your multisig's name.
2. Insert the addresses that will be part of the signing set.
3. Add or remove addresses from the set. The minimum number of addresses for a set is two.
4. Define the threshold of signatures to approve the transaction. The minimum threshold is two.
5. Click on **Create**.
