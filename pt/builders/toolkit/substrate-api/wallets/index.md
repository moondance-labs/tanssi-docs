# Carteiras

Carteiras são ferramentas digitais que permitem aos usuários armazenar, gerenciar e transacionar com segurança seus ativos digitais. Elas armazenam as chaves criptográficas necessárias para acessar ativos digitais e permitem que se executem transações em redes blockchain.

as páginas desta seção fornecem uma visão geral de como começar a usar vários provedores de carteira. Ela aborda como criar e importar contas, conectar-se à sua rede e começar a interagir com ela.
