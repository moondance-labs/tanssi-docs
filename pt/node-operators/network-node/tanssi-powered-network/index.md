- [**Usando Docker**](/pt/node-operators/network-node/tanssi-powered-network/rpc-docker/) - instruções para configurar seu nó em um ambiente conteinerizado para fácil implantação e isolamento
- [**Usando Systemd**](/pt/node-operators/network-node/tanssi-powered-network/rpc-systemd/) - orientação para configurar seu nó como um serviço, permitindo o gerenciamento automatizado e a integração do sistema
