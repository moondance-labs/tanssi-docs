# Onboarding

Um dos principais pilares da Tanssi para construtores é que ela fornece segurança econômica de nível Ethereum para cada rede com tecnologia Tanssi, independentemente de seu TVL, base de usuários ou detalhes do caso de uso. O design da Tanssi oferece aos desenvolvedores um modelo de segurança compartilhado, livrando-os de ter que obter segurança econômica suficiente ou negociar com operadores para executar nós optando por suas redes.

O protocolo Tanssi consegue isso integrando-se a [provedores de segurança externos](/pt/learn/tanssi/external-security-providers/){target=\_blank}, como o [Symbiotic](/pt/learn/tanssi/external-security-providers/symbiotic/){target=\_blank}. Nesse modelo, diferentes [cofres](/pt/learn/tanssi/external-security-providers/symbiotic/#vaults){target=\_blank} associados à Tanssi fornecem segurança econômica, e operadores confiáveis fornecem serviços de validação à rede Tanssi.

Siga estas etapas para configurar seu nó e preparar sua conta para participação ativa no protocolo Tanssi.
